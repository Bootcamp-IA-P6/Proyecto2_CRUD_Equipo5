Vehicle Rental Management System

This project is a backend-focused vehicle rental system built with Django and Django REST Framework. It demonstrates RESTful API design, authentication with JWT, business logic handling, and a basic web interface.

Project Structure
  • API Layer: RESTful endpoints for users, cars, and reservations
  • Authentication: JWT-based authentication for protected resources
  • Business Logic: Automated pricing and validation rules
  • Web Interface: Django templates with Fetch-based API integration
